# Images Folder

Place your images in this folder and name them as follows:

## Gallery Images (4:3 aspect ratio recommended)
- `gallery-1.jpg` — First portfolio image
- `gallery-2.jpg` — Second portfolio image
- `gallery-3.jpg` — Third portfolio image
- `gallery-4.jpg` — Fourth portfolio image
- `gallery-5.jpg` — Fifth portfolio image
- `gallery-6.jpg` — Sixth portfolio image

## Background Image (optional)
- `hero-bg.jpg` — Hero section background (16:9 aspect ratio recommended)

## Tips
- Use JPG or PNG format
- Optimize images for web (try to keep under 500KB each)
- For best results, gallery images should be at least 1200x900px
- Hero background should be at least 1920x1080px
